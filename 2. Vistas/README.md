
# CRUD BÁSICO EN HTML

Elaboración de un crud básico en html, css y js.

## Features  

- Crear (CREATE)
- Leer (READ)
- Actualizar (UPDATE)
- Eliminar (DELETE)

## Elaborarlo en local

Clonar el repositorio

~~~bash  
  git clone https://link-to-project
~~~

Entrar a la carpeta del repositorio

~~~bash  
  cd my-project
~~~

Instalar las dependencias  

~~~bash  
npm install
~~~

Iniciar el servidor

~~~bash  
npm run start
~~~
